# Silen AI 操作指南

使用这份契约搭建、读取或维护 Silen 文档站，无需抓取面向人的网页界面。

## 发现入口

1. 修改本地项目时，优先读取已安装 npm 包中的 Agent Contract。
2. 阅读线上站点时，优先读取该站点生成的 Silen manifest。
3. 根据目标选择同名任务手册。
4. 在编造配置项、命令或 MCP 工具前，先读取结构化 API 契约。

## 权限

MCP 默认只读。只有用户显式使用 --allow-write 启动服务时才会出现写工具。
内容写权限不自动包含构建、提交、推送或部署权限。

## 完成条件

内容修改后运行 AI audit 和生产构建，检查 Git diff，并报告修改文件与验证结果。
