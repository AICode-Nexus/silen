## 为安静的文档而生

Silen 让作者工作流保持轻量：写 Markdown 或 MDX，只在真正需要时加入
React，然后用稳妥的默认配置发布快速的静态站点。

```sh
pnpm add -D silen
pnpm silen dev docs
```

## 一次构建，服务两类读者

每次 Silen 构建都从同一个可信源服务人类读者和 AI 系统。读者得到可访问的
导航、搜索、代码高亮和响应式主题；AI 客户端得到确定性的 Markdown、清单和
可选的本地 MCP 工作区，而不会把服务端密钥暴露到浏览器。
