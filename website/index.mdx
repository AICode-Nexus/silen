## Built for calm documentation

Silen keeps the authoring loop small: write Markdown or MDX, add React only
where it helps, and ship a fast static site with sensible defaults.

```sh
pnpm add -D silen
pnpm silen dev docs
```

## One build, two audiences

Every Silen build serves people and AI systems from the same trusted source.
Readers get accessible navigation, search, code highlighting, and a responsive
theme. AI clients get deterministic Markdown, manifests, and an optional local
MCP workspace—without exposing provider keys in the browser.
