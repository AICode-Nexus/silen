# 翻译提示

将《Silen Model-Free AI Quality Loop Design》从英文翻译为简体中文。

## 风格与读者

- 面向开发者、维护者和架构评审者。
- 使用准确、简洁、规范性的技术文档语言。
- 保持原文逻辑、约束等级、章节编号和信息密度。
- 不添加原文没有的产品能力、解释、建议或承诺。

## 格式要求

- 完整保留 Markdown 层级、列表、代码块与 JSON 结构。
- 命令、路径、字段名、版本号和退出码原样保留。
- 产品名与标准缩写保持英文：Silen、Ask AI、Agent Contract、MCP、CI、MiniSearch。
- 使用 `01-analysis.md` 中的统一术语。
- 标题自然中文化，但文件名和示例 ID 不翻译。
- 不添加译者注，不删除重复但有规范意义的约束。

## 关键语义

- “无模型”必须表达为第一等、完整、默认可用的产品路径。
- Ask AI 必须表达为独立可选增强，不能暗示 Silen 内置真实助手。
- “确定性”必须保留相同输入产生相同输出的工程含义。
- “只读”“不得执行配置”“不得联网”“不修改源码”等安全边界不得弱化。
- “阻断问题”和“非阻断提示”必须清晰区分。
